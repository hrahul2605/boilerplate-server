## Pre-requisites

* Node
* MongoDB
* Redis

## Get Started

1. Create `.env` file and populate it taking reference to `sample.env`.
2. `yarn install` - Install all the dependencies
3. Check your Mongo & Redis server is up and running.
4. `yarn start` - Start the server.
